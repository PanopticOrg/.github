## Panoptic

Panoptic is a software dedicated to the exploration of large image datasets build by and for researchers at https://github.com/CERES-Sorbonne.

This organisation aims to host the main Panoptic repository and the plugins developped by our team.

Want to get started ? you can:
- [Install panoptic](https://github.com/CERES-Sorbonne/Panoptic?tab=readme-ov-file#2-installation)
- [Read how to write plugins](https://github.com/CERES-Sorbonne/Panoptic/wiki/Plugin)
- [Read or cite us (french only for now)](https://inserm.hal.science/HUMANISTICA-2024/hal-04687627v1)
- [Listen to our FOSDEM presentation](https://ceres.sorbonne-universite.fr/ded701fb-3357-4991-be7f-6e218403c4b7/)
